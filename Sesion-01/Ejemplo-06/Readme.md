## üöÄ Comencemos

Los operadores de asignaci√≥n facilitan la actualizaci√≥n de valores de una manera mucho mas limpia y concisa. Estos operadores combinan operaciones aritm√©ticas o l√≥gicas con la asignaci√≥n tradicional, permitiendo modificar el valor de una variable sin la necesidad de repetir su nombre. Este enfoque acelera la escritura del c√≥digo, lo hace m√°s intuitivo y menos propenso a errores, contribuyendo a la claridad del programa.

### üìú Lista de operadores de asignaci√≥n

- `=`: Asignaci√≥n simple.
- `+=`: Asignaci√≥n con suma.
- `-=`: Asignaci√≥n con resta.
- `*=`: Asignaci√≥n con multiplicaci√≥n.
- `/=`: Asignaci√≥n con divisi√≥n.
- `%=`: Asignaci√≥n con m√≥dulo.
- `**=`: Asignaci√≥n con exponente.

### üß† Ejemplos de operadores  de asignaci√≥n

### Asignaci√≥n Simple (`=`)

Asigna un valor directo a una variable.

```python
x = 10  # x ahora es 10
```

### Asignaci√≥n con Suma (`+=`)

Suma un valor al valor de la variable y actualiza la variable con el resultado.

```python
x = 5
x += 3  # x ahora es 8
```

### Asignaci√≥n con Resta (`-=`)

Resta un valor del valor de la variable y actualiza la variable con el resultado.

```python
x = 5
x -= 2  # x ahora es 3
```

### Asignaci√≥n con Multiplicaci√≥n (`*=`)

Multiplica el valor de la variable por un valor y actualiza la variable con el resultado.

```python
x = 4
x *= 3  # x ahora es 12
```

### Asignaci√≥n con Divisi√≥n (`/=`)

Divide el valor de la variable por un valor y actualiza la variable con el resultado.

```python
x = 10
x /= 2  # x ahora es 5
```

### Asignaci√≥n con M√≥dulo (`%=`)

Aplica un m√≥dulo entre el valor de la variable y un valor y actualiza la variable con el resultado.

```python
x = 10
x %= 3  # x ahora es 1
```

### Asignaci√≥n con Exponente (`**=`)

Eleva el valor de la variable a la potencia de un valor y actualiza la variable con el resultado.

```python
x = 2
x **= 3  # x ahora es 8
```

### Asignaci√≥n con Divisi√≥n Entera (`//=`)

Aplica una divisi√≥n entera entre el valor de la variable y un valor y actualiza la variable con el resultado.

```python
x = 10
x //= 3  # x ahora es 3
```
